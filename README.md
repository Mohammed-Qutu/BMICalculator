# BMICalculator

## Description
This is an interactive Java program that allows users to calculate their Body Mass Index (BMI) by entering their height (in inches) and weight (in pounds). The BMI is calculated using a functional method and is displayed to the user.

## Author
Written by Mohammed Qutu on 2/9/2024.

## Requirements
- Java 21.0.2 or later

## Usage
To run the program, compile the `CalculatingBMIExercise.java` file and execute it. The user will be prompted to enter their height and weight, after which the program will calculate and display their BMI.
